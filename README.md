## ğŸ¯ How It Works

1. The program starts with an empty *NÃ—N* chessboard.
2. It attempts to place queens one by one in each column.
3. If a conflict arises, it **backtracks** and tries a new position.
4. This process continues until a valid configuration is found or all possibilities are exhausted.
5. The GUI highlights each move, showing both placements and removals of queens.
